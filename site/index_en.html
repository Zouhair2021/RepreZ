Visual Representation of Numbers

RepreZ is an educational application that allows visualization of numbers from 0 to 9999 using blocks representing thousands, hundreds, tens, and units. This visual representation method is particularly effective for learning the decimal system and place value numeration.

Developed in Java, this application offers an interactive interface allowing teachers and students to generate, represent, and capture visual representations of numbers.

How to Use RepreZ

Simple Representation

1. Enter a number between 0 and 9999 in the text field

2. Click on "Represent" to visualize the number

3. Use "Capture" to save the image in the captures folder

Random Generation

1. Set the minimum and maximum values for the range

2. Click on "Generate" to get a random number within this range

3. The number is automatically represented visually

Automatic Capture

1. Check "Auto Capture" at the top of the interface

2. Indicate the desired number of captures

3. Set the min/max range

4. Click on "Generate" to start the generation and automatic capture
